Hexo的安装与使用

发表于 2019-03-24

Github

1. Create a new repository with name below

Hexo installation

1. Install git and Node.js

2. Install Hexo by npm

3. Create a local folder and init folder

Github Plugin

1. Install github plugin

2. Change _config.yml which is located in local folder root

Theme

1. Enter your local Hexo folder and download theme next

2. Change _config.yml which is located in local folder root

3. Change our favorite Mist Scheme by file yourHexoFolder/themes/next/_config.yml

Now Start Writing !

1. Create a new blog page and deploy

2. Delete a blog page and deploy

  1. Delete the post under source/_post folder
  2. Run hexo clean to delete the database (db.json) and assets folder
  3. Run hexo generateto generate the new blog without your deleted post
  4. Run hexo deploy to deploy your blog
